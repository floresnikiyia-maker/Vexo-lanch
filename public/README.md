# Vexo Launcher Public Assets

This folder contains all public-facing assets for the Vexo Launcher application.

## Contents

- **icon.svg** - Main application icon (vector)
- **icon-192.png** - 192x192 PNG icon for web manifest
- **manifest.json** - Web app manifest for PWA capabilities
- **globals.css** - Global CSS styles and animations

## Usage

All files in this folder are accessible from the root of the application:
- `/icon.svg`
- `/icon-192.png`
- `/manifest.json`
- `/globals.css`

## Adding New Assets

Simply add your files here:
- Images (PNG, JPG, WebP, SVG)
- Fonts (if not using CDN)
- Audio files
- Other static resources

All assets are automatically served by Vite in development and included in the build.
